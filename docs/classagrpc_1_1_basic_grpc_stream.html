Detailed Description

template<class Executor>
class agrpc::BasicGrpcStream< Executor >

Cancellation safety for streaming RPCs.

Lightweight, IoObject-like class with cancellation safety for RPC functions.

inline

Either wait for the initiated operation to complete or complete immediately.

If the initiated operation has already completed then the completion handler will be invoked in a manner equivalent to using asio::post.

Per-Operation Cancellation

All. Upon cancellation, the initiated operation continues to run.
